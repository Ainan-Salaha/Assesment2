HTML Interview Q&A (Day-2)

1.Difference between HTML tags and elements?

Ans:     HTML tags- Tags are the starting and ending part of an HTML element. They begin with “ < ” symbol and end with “ > ”. Whatever written inside < and > are called tags. For instance: -- < b>< /b>

     HTML elements -Elements enclose the contents in between the tags. They consist of some kind of structure. It generally consists of a start tag, content and end tag. For instance: -- < b>Content< /b>

2.Explain br tags?

Ans:     The < br> tag in HTML document is used to create a line break in a text. It is an empty tag which means it does not need end tag like < /b>.

It is generally used in poem and address where the division of line is necessary.

3.Explain how many different types of lists we have in html ?

Ans:     In HTML there are 3types of list
  1. Ordered List
  2. Unordered List
  3. Description List

    1.Ordered List

        In ordered list we have different types of list:-
  1.      Numerical List[1] (1,2,3,4.. etc)
  2.     UpperCase Letter[A] (A,B,C,D..etc)
  3.     LowerCase Letter[a] (a,b,c,d..etc)
  4.     UpperCase Roman Numerals[I] (I,II,III,IIII..etc)
  5.      LowerCase Roman Numerals[i] (i,ii,iii,iiii..etc)

    2.Unordered List

    In unordered list we have different types of list:-
  • Circle type list
  • Disc type list
  • Square type list
  • None type list

    3.Description List

     A Description List is a list of terms,with a description of each term
